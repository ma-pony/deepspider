# DeepSpider

> 智能爬虫工程平台 - 基于 DeepAgents + Patchright

从 JS 逆向分析到完整爬虫脚本，一站式解决方案。

## 特性

- **逆向分析**: Webpack 解包、反混淆、加密算法定位
- **动态调试**: 真实浏览器 + CDP 断点调试
- **代码转换**: JS 加密逻辑自动转 Python
- **验证码处理**: 滑块、点选、图片验证码
- **反检测**: 指纹伪装、代理轮换、风控规避
- **爬虫编排**: 智能调度，输出可运行的 Python 爬虫

## 文档

## 安装

```bash
pnpm install
cp .env.example .env  # 配置 ANTHROPIC_API_KEY
```

## 使用

```bash
pnpm run agent       # Agent 交互模式
pnpm run mcp         # MCP 服务
pnpm test            # 运行测试
```
