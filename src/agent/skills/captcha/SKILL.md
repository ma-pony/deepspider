# 验证码处理经验

## 滑块验证码

### 缺口检测
- 边缘检测：Canny 算法定位缺口边缘
- 模板匹配：滑块图与背景图匹配

### 轨迹生成要点
- 起始慢，中间快，结束慢
- 添加微小随机抖动
- 总时长 300-800ms
- 结束时有轻微回弹

## 图片验证码

### OCR 选择
- ddddocr：开源免费，识别率一般
- 打码平台：付费，识别率高

### 预处理技巧
- 二值化去除背景
- 去噪点
- 字符分割

## 点选验证码

### 目标检测
- 文字定位
- 图标识别
- 顺序判断
